Antes de aprender qualquer linguagem de programação, é essencial compreender a **lógica de programação** — o raciocínio sistemático por trás da criação de algoritmos. A lógica de programação não зависит de uma linguagem específica; é uma forma de pensar que você aplicará em qualquer contexto computacional.

## O que é Lógica de Programação?

Lógica de programação é o processo de criar sequências ordenadas de instruções para resolver problemas. Ela envolve:

- **Decomposição**: Dividir um problema complexo em partes menores
- **Reconhecimento de padrões**: Identificar similaridades entre problemas
- **Abstração**: Focar nos aspectos relevantes, ignorando detalhes desnecessários
- **Algoritmia**: Criar a sequência de passos para resolver cada subproblema

## Estruturas Fundamentais

Toda lógica de programação se baseia em três estruturas básicas:

### 1. Sequência

As instruções são executadas uma após a outra, na ordem em que aparecem.

```
pegue o pão
corte o pão
passe a manteiga
sirva
```

### 2. Seleção (Decisão)

O fluxo de execução muda com base em uma condição.

```
se (estou com fome) então
    coma
senão
    espere
fim-se
```

### 3. Repetição

Um bloco de instruções é executado múltiplas vezes.

```
enquanto (há pratos sujos) faça
    lave um prato
fim-enquanto
```

## Variáveis e Tipos de Dados

Para processar informações, precisamos de **variáveis** — espaços na memória que armazenam valores. Cada variável tem um tipo:

- **Inteiro**: Números sem parte decimal (1, 42, -7)
- **Real**: Números com parte decimal (3.14, -0.5)
- **Caractere**: Um único caractere ('a', 'X')
- **Texto/Cadeia**: Sequências de caracteres ("Olá mundo")
- **Lógico/Boolean**: Verdadeiro ou falso

## Operadores

### Aritméticos
- Soma (+), Subtração (-), Multiplicação (*), Divisão (/)

### Relacionais
- Maior que (>), Menor que (<), Igual (==), Diferente (!=)

### Lógicos
- E (and), OU (or), NÃO (not)

## Fluxogramas

Uma forma visual de representar algoritmos é através de **fluxogramas**, que usam símbolos padronizados:

- **Ovais**: Início e fim
- **Retângulos**: Processos/ações
- **Diamantes**: Decisões
- **Setas**: Fluxo de execução
